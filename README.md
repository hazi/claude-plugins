# claude-plugins

HAZI の Claude Code プラグインマーケットプレイス。

## インストール

```bash
/plugin marketplace add hazi/claude-plugins
```

## プラグイン一覧

### jplint

日本語テキスト校正プラグイン。textlint をベースに、日本語文書の品質チェックと自動修正を行う。

```bash
/plugin install jplint@hazi-plugins
```

詳細は [plugins/jplint/](plugins/jplint/) を参照。
